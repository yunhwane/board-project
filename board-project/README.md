# 게시판 연습 프로젝트 
- **✨ 게시판 Paging 처리한 CRUD project**

## Features
- **✨ Spring Security를 활용한 Login 처리**
- **✨ JWT AccessToken, RefreshToken 발급**
- **✨ 권한에 따른 접근 처리**
- **✨ Custom Filter (JSON, JWT)**
- **✨ validation**
- **✨ Custom Exception, ErrorCode**
- **✨ Redis를 활용한 Refresh Token 저장/조회 구현**
